# LangOver - המרת הקלדה שגויה עברית ↔ אנגלית

תוכנית פשוטה שממירה טקסט שהוקלד בשפה הלא נכונה.

**דוגמה:** `asdf` → `שדגכ` | `שדגכ` → `asdf`

---

## התקנה

### שלב 1 - התקנת AutoHotkey v2
הריצו בטרמינל (PowerShell):
```
winget install AutoHotkey.AutoHotkey
```
או הורידו מ: https://www.autohotkey.com/

### שלב 2 - הפעלה
לחצו דאבל-קליק על הקובץ `langover.ahk`

### שלב 3 (אופציונלי) - הפעלה אוטומטית עם Windows
1. לחצו `Win+R`
2. הקלידו `shell:startup` ולחצו Enter
3. צרו קיצור דרך (Shortcut) לקובץ `langover.ahk` בתיקייה שנפתחה

---

## שימוש

1. **סמנו** את הטקסט השגוי (Ctrl+A או בעזרת העכבר)
2. **לחצו** `ScrollLock`
3. הטקסט יומר אוטומטית ✓

> קיצור המקשים: **ScrollLock** - מקש שכמעט לא בשימוש במקלדות מודרניות

---

## איך זה עובד?

התוכנית מזהה אוטומטית את השפה:
- אם הטקסט מכיל אותיות עבריות → ממיר לאנגלית
- אם הטקסט לא מכיל עברית → ממיר לעברית

כל תו ממופה לפי המקש הפיזי המשותף בשתי השפות.
